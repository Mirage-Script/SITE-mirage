## 🎓 APRENDIZADOS

1. **GSAP é poderoso** - ScrollTrigger, Flip, Draggable, etc.
2. **Tokens de animação centralizam** - Consistência em todo o projeto
3. **Cleanup é crítico** - ScrollTriggers e timelines precisam ser mortos
4. **Performance importa** - GPU acceleration, lazy loading, memory management
5. **Acessibilidade é essencial** - Reduced motion, keyboard nav, screen readers

## 📝 COMO USAR OS COMPONENTES

### Exemplo 1: SplitType Multi-Level
```tsx
import { SplitTypeMultiLevel } from '@/components/effects/SplitTypeMultiLevel';

<SplitTypeMultiLevel
  text="Seu texto aqui"
  level="chars"
  staggerDelay={0.05}
  blurEffect={true}
  parallaxEffect={true}
/>
```

### Exemplo 2: Clip-Path Reveal
```tsx
import { ClipPathReveal } from '@/components/effects/ClipPathReveal';

<ClipPathReveal clipShape="circle">
  <img src="image.jpg" alt="Imagem" />
</ClipPathReveal>
```

### Exemplo 3: Advanced Hover Effect
```tsx
import { AdvancedHoverEffect } from '@/components/effects/AdvancedHoverEffect';

<AdvancedHoverEffect effectType="all">
  <div className="card">Hover me!</div>
</AdvancedHoverEffect>
```
